USTHB Pseudocode Reference

Complete syntax guide for Algorithms & Data Structures I & II — L1 Mathematics, Faculty of Mathematics, USTHB (2024-2025).

Algorithm Structure

Every algorithm follows this skeleton:

Algorithm Name;
Var
    x, y : integer;
    result : float;
Begin
    // your statements here
End;
Rules: The Algorithm keyword starts the program. Variables are declared in the Var block before Begin. Functions and structs are defined before the main algorithm.

Input / Output

Write("Enter a number: ");
Read(x);
Write("The value is: ", x);
Use commas inside Write() to concatenate multiple values.

Conditions

If — Else

If (x > 0) Then
    Write("Positive");
Else
    Write("Non-positive");
End If;

Switch

Switch (day)
    Case 1: Write("Monday");
    Case 2: Write("Tuesday");
    Else    Write("Other");
End Switch;

Loops

For loop

For i <- 1 to 10 Do
    Write(i);
EndFor;

While loop

While (i <= 10) Do
    Write(i);
    i <- i + 1;
EndWhile;

Do-While loop

Do
    Write(i);
    i <- i + 1;
While (i <= 10);

Repeat-Until

Repeat
    Read(x);
Until (x > 0);

Arrays

Arrays are 1-indexed at USTHB.

// 1D Array
Var A[10] : integer;

For i <- 1 to 10 Do
    Read(A[i]);
EndFor;

// 2D Array (Matrix)
Var M[3][3] : float;
M[1][1] <- 3.14;
Important: indices start at 1, not 0.

Functions & Procedures

Defined before the main Algorithm block.

Function (with return value)

integer function FindMax(a: integer, b: integer)
Begin
    If a > b Then
        Return a;
    Else
        Return b;
    End If;
End;

Procedure (no return value)

void function PrintMessage(msg: string)
Begin
    Write(msg);
End;

Recursive function

integer function Factorial(n: integer)
Begin
    If n = 0 Then
        Return 1;
    Else
        Return n * Factorial(n - 1);
    End If;
End;

Structs (Records)

Defined before the algorithm. Groups related fields under one name.

struct Student {
    name  : string;
    grade : integer;
    gpa   : float;
};

Algorithm Main;
Var
    s1 : Student;
    students : array[1..50] of Student;
Begin
    s1.name  <- "Khaled";
    s1.grade <- 18;
    Write(s1.name, " → ", s1.grade);
End;

Enumerations

enumeration Days { Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday }

void function CheckDay(day: Days)
Begin
    If day = Days.Saturday OR day = Days.Friday Then
        Write("Weekend!");
    Else
        Write("Weekday.");
    End If;
End;

Algorithm Main;
Var today : Days;
Begin
    today <- Days.Monday;
    CheckDay(today);
End;

Pointers

Declaration & basic usage

Var
    x   : integer;
    ptr : *integer;
Begin
    x   <- 42;
    ptr <- &x;           // ptr holds address of x
    Write(*ptr);         // dereference → 42
    *ptr <- 100;         // modify x through pointer
    Write(x);            // → 100
End;

Pass-by-reference via pointers

void function Swap(a: *integer, b: *integer)
Var temp : integer;
Begin
    temp <- *a;
    *a   <- *b;
    *b   <- temp;
End;

Algorithm Main;
Var x, y : integer;
Begin
    x <- 10;
    y <- 25;
    Swap(&x, &y);
    Write("x=", x, " y=", y);   // x=25 y=10
End;

Arrays as pointers

Var
    v : integer[5];
    p : *integer;
Begin
    v <- [1, 2, 3, 4, 5];
    For (p <- v to p + 4) Do
        Write(*p);
    EndFor;
End;
